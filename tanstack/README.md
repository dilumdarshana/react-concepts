# TanStack

Learning project for TanStack libraries with React 19, Vite 8, Tailwind CSS v4, and TypeScript 6.

## Stack

| Library | Purpose |
|---------|---------|
| `@tanstack/react-query` | Server state management (fetch, cache, sync) |
| `@tanstack/react-router` | File-based routing with type safety |
| `@tanstack/react-table` | Headless UI for data tables |
| `@tanstack/react-router-devtools` | Router debugging tools |

## What's inside

### TanStack Query (`/useQuery`, `/useSuspenseQuery`, `/useQueries*`)

| Route | Feature |
|-------|---------|
| `/useQuery` | Basic fetch with `useQuery`, pending/error states |
| `/useQueryWIthParams` | Dynamic query with route params via `useParams` |
| `/useSuspenseQuery` | Suspense-compatible data fetching (guaranteed data) |
| `/useQueriesSync` | Multiple related queries run in parallel via `useQueries` |
| `/useQueriesAsync` | Independent queries fetched concurrently |

Query options are defined in `src/queryOptions/` as composable factories (`createTodoQueryOptions`, `createUserQueryOptions`, etc.).

### TanStack Table (`/tanStackTable`, `/tanStackTableDynamic`)

| Route | Approach |
|-------|----------|
| `/tanStackTable` | Loads full dataset client-side; supports pagination, sorting, and filtering |
| `/tanStackTableDynamic` | Server-side pagination — fetches chunks on demand via `manualPagination` |

### Layout

- `src/layouts/Layout.tsx` — root layout (not used; routing uses `__root.tsx`)
- `src/routes/__root.tsx` — actual root route with sidebar nav (`LeftMenu`) and footer

## Commands

```bash
pnpm dev       # Start dev server (Vite)
pnpm build     # tsc -b && vite build
pnpm lint      # ESLint flat config
pnpm preview   # Vite preview
```

## Notes

- `src/routeTree.gen.ts` is auto-generated by `@tanstack/router-plugin` — never edit manually.
- Tailwind v4 via `@tailwindcss/vite` plugin (no `postcss.config.js` / `tailwind.config.js`).
